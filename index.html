Cara Kerja Internet Secara Umum

Internet adalah jaringan komputer yang saling terhubung dan dapat berkomunikasi satu sama lain.

Internet menggunakan protokol TCP/IP untuk mengirimkan data antar komputer.

Sumber referensi: Wikipedia - Internet, HowStuffWorks - How the Internet Works

HTTP dan HTTPS

HTTP (Hypertext Transfer Protocol) adalah protokol yang digunakan untuk mengirimkan data antar komputer melalui internet.

HTTPS (Hypertext Transfer Protocol Secure) adalah versi aman dari HTTP yang menggunakan enkripsi untuk melindungi data.

Sumber referensi: Wikipedia - HTTP, Wikipedia - HTTPS, Mozilla Developer Network - HTTP dan HTTPS

Nama Domain

Nama domain adalah alamat unik yang digunakan untuk mengidentifikasi sebuah situs web atau server.

Nama domain diubah menjadi alamat IP oleh DNS (Domain Name System).

Sumber referensi: Wikipedia - Nama Domain, ICANN - Domain Name System

Hosting

Hosting adalah layanan yang menyediakan ruang penyimpanan dan akses ke server untuk menyimpan dan mengakses situs web.

Pengguna mengupload file situs web ke server hosting, server hosting menyediakan akses ke situs web melalui internet.

Sumber referensi: Wikipedia - Hosting, HostGator - Bagaimana Hosting Bekerja

DNS

DNS (Domain Name System) adalah sistem yang mengubah nama domain menjadi alamat IP.

DNS menggunakan sistem hierarki untuk mengubah nama domain menjadi alamat IP.

Sumber referensi: Wikipedia - DNS, ICANN - Domain Name System

Browser

Browser adalah perangkat lunak yang digunakan untuk mengakses dan menampilkan situs web.

Browser menggunakan HTTP/HTTPS untuk mengirimkan permintaan ke server dan menerima respon.

Sumber referensi: Wikipedia - Browser, Mozilla Developer Network - Bagaimana Browser Bekerja
